# Lüneburg Info Web App

Diese Web App zeigt die aktuelle Uhrzeit, das Wetter in Lüneburg und ein aktuelles Event an.

## Features
- Echtzeit-Uhrzeit
- Wetterdaten von Open-Meteo
- Statische Event-Daten (können manuell aktualisiert werden)
